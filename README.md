# NDA: Nonparametric Data Attribution for Diffusion Models

This repository is the official implementation of _NDA:Nonparametric Data Attribution for Diffusion Models._


### Setup
Follow the steps below to get started.

1. **Clone the GitHub Repository**
   ```shell
   git clone https://github.com/sail-sg/NDA.git
   ```
2. **Create a Python 3.9 environment**
   ```shell
   conda create -n nda python=3.9 -y
   conda activate nda
   ```
3. **Install Dependencies**
   ```shell
   pip install -r requirements.txt
   ```
